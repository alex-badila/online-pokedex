# Online Pokedex

See the site live: https://web301-assignment-3.vercel.app

## Description

This site shows the information of the first 151 Pokemon. The home page lists each Pokemon's name in a card. Once a card is clicked, the page changes to display an image of the Pokemon in question, along with its information. The loading icon is a spinning Poke Ball. 

This site was the third and final assignment for the JavaScript and ReactJS course as part of Seneca's Web Programming and Development program.
